## General info

PCS Notebook is a simple application for taking notes.

## Features

* Save notes in database
* Search notes
* Minimalist design

## Technologies

This project was created with:

* Qt version: >4

## Setup

```
$ mkdir -p build
$ cd build
$ qmake ../src/pcsnotebook.pro
$ make
$ make install
```
