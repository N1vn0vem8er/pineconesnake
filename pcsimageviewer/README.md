## General Info

PCS Image Viewer is a simple image viewer.

## Features

* Opens images
* Remembers recently opened images
* Can cycle through images in directory with arrow key
* Minimalist design

## Technologies

This project was created with:

* Qt version: >4

## Setup

```
$ mkdir -p build
$ cd build
$ qmake ../pcsimageviewer.pro
$ make
$ make
```
