## General info

PCS Messages Archiver is an application for manual saving of messages into database.

## Features

* Save messages
* Browse messages
* Minimalist design

## Technologies

This project was created with:

* Qt version: >4
* SQLite3 version: >=3.46.1

## Setup

```
$ mkdir -p build
$ cd build
$ qmake ../src/pcsmessagesarchiver.pro
$ make
$ make install
```
