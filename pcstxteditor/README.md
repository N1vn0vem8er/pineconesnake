## General info

PCS TxtEditor is a basic text files editor.

## Features

* Opens multiple files in tabs
* Line numbering
* Minimalist design

## Technologies

This project was created with:

* Qt version: >4

## Setup

```
$ mkdir -p build
$ cd build
$ qmake ../src/pcstxteditor.pro
$ make
$ make install
```
